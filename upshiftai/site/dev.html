Developers

Integrate dependency intelligence into your workflows. CLI, CI/CD, and programmatic API.

Installation

Zero Install (npx)

$ npx upshiftai-deps analyze .

Global Install

$ npm install -g upshiftai-deps

Programmatic API

Use UpshiftAI as a library in your own Node.js tools.

script.js
import { analyze, applyNpmUpgrade } from 'upshiftai-deps';

// 1. Analyze a project
const report = await analyze('./frontend', {
  fetchRegistry: true,
  ancientMonths: 24
});

console.log(`Found ${report.summary.ancient} ancient deps`);

// 2. Upgrade a package
await applyNpmUpgrade('./frontend', 'lodash', 'latest');

CI Integration

Fail builds if dependencies are too old or have security vulnerabilities.

.github/workflows/audit.yml
name: Dependency Audit
on: [push]

jobs:
  upshiftai:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - run: npx upshiftai-deps health . --exit-code

Configuration

Create .upshiftai.json in your project root.

{
  "webhooks": ["https://api.your-company.com/hooks/deps"],
  "approval": {
    "mode": "prompt",
    "requireFor": ["replace", "major"]
  }
}
