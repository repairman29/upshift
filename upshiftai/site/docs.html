Documentation

Learn how to analyze dependency trees, fix ancient packages, and automate security workflows with UpshiftAI v0.2.1.

Core Concepts

1. Analyze

Resolves full trees (npm/pip/Go), fetches registry metadata, and runs security audits.

2. Report

Generates "deep throat" reports with transitive chains and risk assessments.

3. Fix

Automated upgrades and replacements with checkpoint safety.

4. Safety

Checkpoints save state before changes; rollbacks restore it instantly.

CLI Reference

Run npx upshiftai-deps@0.2.1 or install globally.

Analysis & Reporting

# JSON analysis with security audit
$ npx upshiftai-deps analyze . --summary --exit-code
# Full report (Markdown/PDF)
$ npx upshiftai-deps report . --pdf --licenses

Automated Fixes (npm + pip)

# Batch fix ancient deps
$ npx upshiftai-deps apply fix . --dry-run --yes
# One-shot package fix
$ npx upshiftai-deps fix moment . --yes

Signals & Risk

Signal Meaning
age Months since last publish (>24 months = ancient)
deprecated Marked deprecated in registry (npm/PyPI)
audit Known vulnerabilities (npm audit / pip-audit)
fork_hint Package name suggests a fork (-legacy, -fork)
