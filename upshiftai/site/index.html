Try it (CLI)

Everything is executable from the CLI. From the upshiftai directory or via npx upshiftai-deps:

# Analyze (JSON / markdown / CSV)
upshiftai-deps analyze [path] [--markdown] [--csv] [--no-registry] [--ecosystem=npm|pip|go]

# Full report (direct + transitive, ‚Äúsomething old‚Äù chains, optional --pdf)
upshiftai-deps report [path] [--output FILE] [--pdf] [--no-full-tree]

# Checkpoint before changes, rollback to restore
upshiftai-deps checkpoint [path] [--reason "..."]
upshiftai-deps rollback [path] [--dry-run]

# Apply automated fixes (npm AND pip; checkpoint, verify, auto-rollback)
upshiftai-deps apply fix [path] [--dry-run] [--yes] [--limit N]
upshiftai-deps fix <pkg> [path] [--dry-run] [--yes]

# Health check (OK/WARN/FAIL status for CI)
upshiftai-deps health [path] [--exit-code] [--json]

Use .upshiftai.json (see .upshiftai.example.json) for webhooks and approval policy. Full reference: How it works.
