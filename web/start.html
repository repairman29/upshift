Get Started

Install Upshift and scan your first project in under a minute.

1

Install the CLI

Requires Node.js 18 or higher.

npm install -g upshift-cli
2

Scan your project

Navigate to any project with a package.json and run:

upshift scan

Upgrade safely

Upshift upgrades the package, runs tests, and rolls back if anything fails:

upshift upgrade react # or preview first: upshift upgrade react --dry-run

Free vs AI-powered

upshift explain is free and unlimited. Add --ai for deep AI analysis (costs 1 credit). You get 10 free credits to start.

Credit packs (for AI features)

  • upshift buy-credits --pack small ‚Üí 100 credits for $5
  • upshift buy-credits --pack medium ‚Üí 300 credits for $15
  • upshift buy-credits --pack large ‚Üí 1000 credits for $40

Subscriptions (includes monthly credits)

  • upshift subscribe --tier pro ‚Üí $9/mo (100 credits included + 20% bonus on packs)
  • upshift subscribe --tier team ‚Üí $29/mo (500 credits included + 20% bonus)
