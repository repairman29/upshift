# Upgrading Express from 4.18.2 to 5.0.0

## Breaking Changes

1. **Response methods changed**
   - `res.send()` no longer accepts objects directly
   - Use `res.json()` for JSON responses instead

2. **Body parser removed**
   - Built-in `bodyParser` middleware is removed
   - Use `express.json()` and `express.urlencoded()` instead

3. **Error handling signature**
   - Error handlers now require exactly 4 parameters
   - Must be `(err, req, res, next)` - all four required

4. **Path route matching**
   - Stricter path matching in routes
   - Trailing slashes are now significant
